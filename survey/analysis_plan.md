## 2. 분석 1: 교차문화 보편성 검증 (RQ1) [확인적]

### 가설 (수정)
- **H0 (영가설)**: 국가간 종이 전투 놀이 경험률에 유의미한 차이가 있다 (|Δ| > 동등성 마진).
- **H1 (연구가설)**: 국가간 종이 전투 놀이 경험률 차이는 동등성 마진(±10%p) 이내이다 — 즉, 문화에 관계없이 유사한 비율로 경험한다.

### 1차 분석: TOST 동등성 검정 (Two One-Sided Tests)
- **목적**: 국가간 경험률이 "유의미하게 다르지 않음"을 적극적으로 증명
- **방법**: 각 국가 쌍에 대해 TOST 실시
  - 동등성 마진: Δ = ±10%p (사전등록)
  - α = 0.05
  - 두 one-sided test가 모두 유의할 때 동등성 결론
- **보고**: 동등성이 확인된 국가 쌍의 수 / 전체 쌍의 수
- **시각화**: 국가별 경험률 forest plot (동등성 마진 밴드 포함)

### 대안 분석: 베이지안 분석
- **목적**: "차이 없음" 모델 vs "차이 있음" 모델을 직접 비교
- **방법**: 베이지안 로지스틱 회귀
  - M0 (null): 경험유무 ~ 1 (국가 효과 없음)
  - M1 (alternative): 경험유무 ~ 국가
  - Bayes Factor (BF01) 계산
- **판정 기준**:
  - BF01 > 3: 차이 없음 모델 지지
  - BF01 > 10: 차이 없음 모델 강하게 지지
  - 1/3 < BF01 < 3: 판단 보류
  - BF01 < 1/3: 차이 있음 모델 지지
- **보고**: BF01 값 + 사후 분포 플롯

### 보충 분석: 카이제곱 검정 [탐색적]
- **국가 × 경험 유무 카이제곱 독립성 검정**: 탐색적 참고용으로만 보고
- Cramer's V 효과 크기 보고
- 주의: 유의미한 차이가 나와도 TOST/BF 결과와 함께 해석

### 로지스틱 회귀 [확인적]
```
경험유무 ~ 국가 + 성별 + 출생코호트 + 성장환경 + 표집유형(Prolific vs 대학네트워크)
```
- 오즈비(OR) + 95% CI 보고
- 국가 효과의 크기가 작을수록 보편성 근거 강화
- 표집유형 변수로 Prolific vs 대학 네트워크 간 차이 통제

### 시각화
- 국가별 경험률 forest plot (동등성 마진 ±10%p 밴드)
- 세계 지도 히트맵 (경험률 색상 코딩)
- Bayes Factor 막대 그래프

## 3. 분석 2: 이중 비율 분석 — 자발적 회상 vs 유도 재인 [확인적]

- **갭(gap)**: 유도 재인률 - 자발적 회상률 = 잠재적 과소보고 추정치
- 국가별 갭 크기 비교 (문화적 기억 접근성 차이)

### 시각화
- 국가별 이중 막대 그래프 (자발적 회상 vs 유도 재인, 갭 표시)

## 6. 분석 5: 측정 동등성 검증 (Measurement Invariance) [확인적]

### 목적
- 리커트 척도 문항(B7 규칙 체계, B12 향수, C4 보편성 인식)이 언어/문화권에 걸쳐 동일한 구성개념을 측정하는지 검증
- 교차문화 비교의 전제 조건 확인

### 방법: 다집단 확인적 요인분석 (Multi-Group CFA)
- **단계적 불변성 검증**:
  1. **형태 불변성 (Configural)**: 동일 요인 구조
  2. **약 불변성 (Metric)**: 요인 부하량 동일
  3. **강 불변성 (Scalar)**: 절편 동일
  4. **엄격 불변성 (Strict)**: 잔차 분산 동일
- 최소 약 불변성(metric invariance) 확보 시 국가간 평균 비교 허용
- ΔCFI ≤ 0.01, ΔRMSEA ≤ 0.015 기준 (Chen, 2007)

### 보고
- 각 단계별 적합도 지수 (CFI, RMSEA, SRMR)
- 불변성 미충족 시: 부분 불변성(partial invariance) 탐색 후 해당 국가/문항 명시

## 9. 분석 8: 저경험률 국가 대응 규칙 (Decision Rules for Low-Prevalence Countries)

### 사전 정의된 결정 규칙
- **시나리오 A**: 모든 국가에서 유도 재인률 ≥ 40%
  → 보편성 주장 지지. TOST + BF로 동등성 확인.
- **시나리오 B**: 1-2개국에서 유도 재인률 < 30%
  → 해당 국가를 "저경험률 국가"로 분류.
  → Phase 3 인터뷰에서 해당 국가 과잉표집(oversample).
  → 보편성 주장을 "대부분의 문화에서 보편적"으로 하향 조정.
  → 해당 국가의 놀이 환경/문화적 맥락을 질적으로 분석.
- **시나리오 C**: 3개국 이상에서 유도 재인률 < 30%
  → 보편성 주장 재검토.
  → "보편적 충동의 문화적 조건" 프레임으로 전환.
  → 어떤 문화적 조건에서 이 놀이가 발현/억제되는지를 주요 질문으로 재설정.
- **자발적 회상률과 유도 재인률의 갭이 큰 국가**:
  → 문화적 기억 접근성 차이 논의.
  → 놀이는 존재하나 명명/범주화되지 않았을 가능성 탐색.

## 12. 검정력 분석 (Power Analysis)

### TOST 동등성 검정 (1차 분석)
- 동등성 마진: Δ = ±10%p
- α = 0.05, power = 0.80
- **국가당 n ≈ 100명 필요** (경험률 ~50% 가정)
- 핵심 5개국 × 100명 = N = 500

### 베이지안 분석 (대안)
- BF01 > 3 판정을 위한 시뮬레이션 기반 표본 크기 산정
- 국가당 n = 80-100에서 충분한 정밀도

### 보충국
- 국가당 n = 80 × 3개국 = 240명

### 총 목표
- **N = 1,000+ (핵심 500 + 보충 240 + 여유분 260)**

### 참고 (기존 카이제곱 검정력 — 보충용)
- 효과 크기: w = 0.2 (medium)
- df = 7 (8개국 - 1)
- α = 0.05, power = 0.80
- 필요 N ≈ 253

## 13. 보고 기준

- APA 7th edition 형식
- 효과 크기 반드시 보고 (Cohen's d, Cramer's V, OR, BF)
- 95% 신뢰구간 보고 (빈도주의); 95% 신용구간 보고 (베이지안)
- p-value + 효과 크기 동시 보고 (p-value만으로 판단하지 않음)
- 다중비교 보정: Bonferroni 또는 Holm
- TOST 결과: 90% CI가 동등성 마진 이내에 있는지 그래프로 제시
- BF 결과: BF01 값 + 해석 + 사후 분포 시각화
- 확인적/탐색적 분석 구분 명시
- 개방형 응답은 원문 + 영어 번역 병기
