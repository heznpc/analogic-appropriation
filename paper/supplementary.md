# Supplementary Materials

## Analogic Appropriation: Children's Participatory Character Combat as Cross-Culturally Recurrent Play Across Three Media Eras

## S1. Survey Methodology

### S1.1 Survey Overview

| Item | Detail |
|------|--------|
| Target population | Adults aged 18--45 (retrospective recall of childhood play) |
| Target N | 1,000+ (core countries: 100/country; supplementary countries: 80/country) |
| Core countries (Prolific) | Korea, United States, Japan, Germany, Brazil |
| Supplementary countries (university network) | Indonesia, India, Turkey |
| Languages | Korean, English, Japanese, German, Portuguese (Brazilian), Indonesian, Hindi/English bilingual, Turkish |
| Translation protocol | Expert forward-back translation + cognitive interviews (5/language) |
| Estimated duration | 8--12 minutes |
| Distribution | Prolific (5 core countries) + local university partner networks (3 supplementary countries) |
| Platform | Qualtrics |
| Compensation | Prolific: $2.50/participant; university network: locally equivalent value |

#### Country Selection Rationale

- **Russia excluded**: Prolific panel unavailable due to sanctions
- **Mexico excluded**: Insufficient Prolific panel size
- **Supplementary country caveat**: University network convenience samples; sampling method differences noted in interpretation

### S1.2 Mixed-Methods Design (Exploratory Sequential)

**Phase 1: Exploratory Qualitative Pilot**
- Sample: n=10/country (total N=80)
- Method: Open-ended online survey
- Purpose: Identify country-specific play variants; assess cultural validity of items
- Output: Culturally validated main survey instrument

**Phase 2: Main Survey**
- Sample: N=1,000+ (core: 100/country; supplementary: 80/country)
- Method: Structured survey refined by Phase 1 results
- Distribution: Prolific (5 core countries) + university networks (3 supplementary countries)

**Phase 3: Purposive Interview Sampling**
- Sample: N=30+ (purposive sampling based on survey results)
- Sampling criteria: Oversample low-prevalence countries; prioritize unusual variant reporters
- Three cohorts: Notebook generation (born 1985--2000), Roblox generation (born 2005--2012), Brainrot generation (born 2010--2013)
- Method: Semi-structured, 60 minutes, online

**Integration Points:**
1. Phase 1 (qual) --> Phase 2 (quan): Play variants identified in qualitative pilot incorporated into B6 response options
2. Phase 2 (quan) --> Phase 3 (qual): Low-prevalence countries and unusual response patterns from survey inform interview sampling
3. Phase 3 (qual) --> Interpretation: Interview data provide contextual explanations for statistical patterns

### S1.3 Survey Instrument

#### Section A: Demographics (5 items)
1. Age (birth year)
2. Gender
3. Nationality / country of upbringing
4. Childhood environment (urban / suburban / rural)
5. Primary childhood play setting (school / home / playground / PC room / other)

#### Section B: Core -- Paper Combat Play Experience (14 items)

**B1. Two-Stage Screening (required)**

> Stage 1 (Unprompted free recall):
> "Please freely describe any games you played with friends using paper during elementary school."
> (Open-ended, no examples provided)

> Stage 2 (Prompted recognition):
> *(Presented after Stage 1 response)*
> "Please select any of the following you have done:" (multiple select)
> - Drawing characters on paper with stats for battle
> - Folding paper and attacking opponent units with dots/lines
> - Flicking pencils/pens to hit opponent drawings
> - Creating RPG worlds/shops/hunting grounds in a notebook
> - Drawing sequential battle scenes like manga/comics
> - Drawing territories/empires on graph paper for war
> - Drawing armies on paper and folding to transfer marks
> - None of the above
> - Cannot remember

**B2. Frequency** (if any selected in B1 Stage 2)
> How often did you do this?
> - Almost daily / Several times a week / Several times a month / Occasionally (a few times a year) / Once or twice

**B2a. Memory confidence**
> How confident are you in the above answer? (5-point Likert: Not at all confident -- Very confident)

**B3. Age range** (multiple select): 5--7 / 8--10 / 11--13 / 14+

**B4. Location** (multiple select): School classroom (during class) / School classroom (break time) / Home / After-school / Other

**B5. Group size**: Alone / 2 / 3--5 / 6+

**B6. Play type** (multiple select): Stat-based character battle / Fold-and-dot attack / Pencil flicking / Notebook RPG / Sequential battle drawing / Grid-paper territory war / Fold-and-transfer army / Other

**B7. Rule complexity**: No rules / Simple rules (rock-paper-scissors) / Systematic rules (damage calculation, turn-based) / Very elaborate rules (entire notebook as rulebook)

**B8. Role differentiation**: Yes, one person served as operator/referee / No, all equal participants / Took turns as operator

**B9. Inspiration source** (multiple select): TV animation/manga / Video games / Movies / Watching friends / Self-invented / Other

**B10. Media interface imitation**: Yes (which game/media?) / No, original format / Cannot remember

**B11. Digital transition** (multiple select): Made games in Roblox/Minecraft / Transitioned to Flash/web games / Mobile apps / Did not transition to digital / Other

**B12. Nostalgia**: 7-point Likert (1 = No nostalgia at all -- 7 = Very strong nostalgia)

#### Section C: Digital Era Comparison (5 items)

**C1. Current children observation**: Yes, on paper / Yes, digitally / Yes, via AI / No / N/A

**C2. Italian Brainrot awareness**: Know it well / Heard of it / Don't know

**C3. Connection perception** (if aware of brainrot): Very similar / Somewhat similar / Not very similar / Completely different

**C4. Universal impulse**: 7-point Likert (1 = Not at all universal -- 7 = Very universal)

**C5. Cross-cultural experience awareness**: Yes (which country/game?) / No / Never thought about it

**C6. Free description** (open-ended, optional)

## S2. Analysis Plan

### S2.1 Pre-registration
- Platform: OSF (Open Science Framework)
- Timing: After Phase 1 pilot completion, before Phase 2 main survey
- Contents: Hypotheses, primary/secondary analysis distinction, equivalence margins, sample size justification, exploratory analysis scope
- Each analysis below marked as [Confirmatory] or [Exploratory]

### S2.2 Data Preprocessing

**Quality filtering:**
- Completion time < 3 min --> remove (rushed)
- Completion time > 30 min --> review (possible abandoned-and-resumed)
- All Likert items identical --> remove (straight-lining)
- Nonsensical text in C6 --> flag
- Extreme inconsistency between B1 Stage 1 and Stage 2 --> flag and review

**Variable coding:**
- Birth year --> generational cohort: Gen X (1975--1980), Early Millennial (1981--1990), Late Millennial (1991--1999), Gen Z (2000--2008)
- Country --> sampling type: Core (Prolific) vs. Supplementary (university network)
- Country --> cultural cluster: East Asia (Korea, Japan), North America (USA), Europe (Germany), South America (Brazil), South/West Asia (India, Turkey), Southeast Asia (Indonesia)
- B1 response --> dual coding: spontaneous recall (mentioned paper combat in Stage 1) and prompted recognition (selected one or more in Stage 2)

### S2.3 Analysis 1: Cross-Cultural Universality (RQ1) [Confirmatory]

**Hypotheses:**
- H0: Significant differences exist in paper combat play prevalence across countries (|delta| > equivalence margin)
- H1: Cross-country prevalence differences fall within the equivalence margin (+-10 percentage points)

**Primary: TOST Equivalence Testing**
- Pairwise TOST for each country pair; equivalence margin delta = +-10pp; alpha = 0.05
- Report: number of equivalent country pairs / total pairs
- Visualization: Forest plot with equivalence margin bands

**Alternative: Bayesian Analysis**
- Bayesian logistic regression: M0 (prevalence ~ 1) vs. M1 (prevalence ~ country)
- Bayes Factor (BF01): >3 supports no-difference model; >10 strongly supports; 1/3--3 inconclusive; <1/3 supports difference model

**Supplementary: Chi-square test** [Exploratory]
- Country x experience chi-square independence test; Cramer's V effect size

**Logistic regression** [Confirmatory]
```
experience ~ country + gender + birth_cohort + childhood_environment + sampling_type
```
- Odds ratios + 95% CI; small country effects strengthen universality argument

### S2.4 Analysis 2: Dual Prevalence -- Spontaneous Recall vs. Prompted Recognition [Confirmatory]

- Spontaneous recall rate: proportion mentioning paper combat in B1 Stage 1
- Prompted recognition rate: proportion selecting one or more in B1 Stage 2
- Gap: recognition rate - recall rate = estimated underreporting
- Cross-country gap comparison (cultural memory accessibility differences)

### S2.5 Analysis 3: Play Type Cultural Variation (RQ1 supplement) [Exploratory]

- Multiple-response crosstab: B6 (play type) x country
- Latent Class Analysis (LCA): 2--6 class solutions; BIC/AIC/entropy criteria
- Expected classes: "Notebook RPG," "Physical combat," "Territorial strategy"

### S2.6 Analysis 4: Media Remediation Patterns (RQ2) [Exploratory]

- H2: 2D RPG peak era (born 1998--2008) cohorts show higher "notebook RPG / interface imitation" rates
- Crosstab: birth cohort x B6 x B10 (interface imitation)
- Qualitative coding of B10 open responses (expected: Korea --> MapleStory; USA --> Pokemon; Japan --> Dragon Quest)

### S2.7 Analysis 5: Measurement Invariance [Confirmatory]

- Multi-group CFA: Likert items (B7, B12, C4) across language/culture groups
- Sequential invariance: configural --> metric --> scalar --> strict
- Criteria: delta-CFI <= 0.01, delta-RMSEA <= 0.015 (Chen, 2007)
- Estimator: WLSMV (ordinal indicators)
- Minimum metric invariance required for cross-country mean comparisons

### S2.8 Analysis 6: Digital Transition Patterns (RQ3) [Exploratory]

- Multiple-response analysis: B11 (digital transition) x birth cohort
- Crosstab: B11 x C2 (current children observation)
- Logistic regression: digital_transition ~ birth_cohort + country + paper_play_frequency

### S2.9 Analysis 7: Nostalgia and Cross-Cultural Perception [Exploratory]

- Descriptives: B12 (nostalgia intensity), B2a (memory confidence)
- Correlations: nostalgia x frequency, nostalgia x rule elaborateness, confidence x frequency
- ANOVA: nostalgia by country
- Regression: cross_cultural_awareness (C5) ~ own_experience + brainrot_awareness (C3) + country

### S2.10 Analysis 8: Decision Rules for Low-Prevalence Countries

- **Scenario A** (all countries recognition >= 40%): Universality supported; confirm with TOST + BF
- **Scenario B** (1--2 countries recognition < 30%): Classify as low-prevalence; oversample in Phase 3 interviews; downgrade claim to "prevalent in most cultures"
- **Scenario C** (3+ countries recognition < 30%): Revisit universality claim; reframe as "cultural conditions of the universal impulse"
- **Large recall-recognition gap**: Discuss cultural memory accessibility; explore possibility of unnamed/uncategorized play

### S2.11 Analysis 9: Qualitative Analysis -- Free Description (C6) and B1 Stage 1 [Exploratory]

- Thematic analysis following Braun and Clarke (2006): familiarization, initial coding, theme searching, theme reviewing, defining/naming, reporting
- Expected themes: "secretly during class," "power dynamics of the operator role," "direct interface copying," "rule disputes as social learning," "adult reinterpretation as TRPG"

### S2.12 Power Analysis

**TOST equivalence testing (primary):**
- Equivalence margin: delta = +-10pp; alpha = 0.05; power = 0.80
- Required: n ~= 100/country (assuming ~50% prevalence)
- Core: 5 countries x 100 = N=500

**Bayesian analysis:**
- BF01 > 3 target: n=80--100/country sufficient

**Supplementary countries:** 3 x 80 = 240

**Total target: N = 1,000+** (core 500 + supplementary 240 + buffer 260)

### S2.14 Reporting Standards

- APA 7th edition format
- Effect sizes mandatory (Cohen's d, Cramer's V, OR, BF)
- 95% confidence intervals (frequentist); 95% credible intervals (Bayesian)
- p-value + effect size jointly reported
- Multiple comparison correction: Bonferroni or Holm
- TOST: 90% CI within equivalence margin (graphical)
- BF: BF01 value + interpretation + posterior distribution visualization
- Confirmatory/exploratory analysis distinction explicit
- Open-ended responses: original language + English translation
